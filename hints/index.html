Introduction

What are SquigglyClues

The essence of interactive fiction is picking up and carrying around small objects, often for no apparent reason. But we assume you can handle that part without any help. For the other bits—the puzzle-y bits—you might want the odd hint here and there.

So...here are some odd hints. They're organized by subject matter, and then subdivided into specific questions and their answers. The questions are phrased to be as spoiler-free as possible, and some questions are red herrings. So in theory you won't necessarily spoil much by just scanning through the list of questions.

How to use this booklet

This is not a booklet, it's a web page, so it is officially our position that we have no idea what you're talking about.

But since that's not very helpful and the whole point of this exercise is to be helpful: If you're stuck at some point in PLACEHOLDER NAME, find a question that seems to be relevant to where you're stuck. Uncap the marker and run it over the first hint. If you did not get a marker with your copy of the booklet, use your mouse: move the cursor into the first hint, left click and hold, and then make a bunch of squiggly motions with the cursor to develop the hint. If that doesn't make sense just try on the sample question below.

If the first hint doesn't help, develop the next hint below it. The hints get more explicit as you go lower, which sounds vaguely smutty but isn't. If you reach the bottom and it still hasn't solved your problem, the next step is complaining about the game on the internet. If that doesn't help, comments, complaints, and bug reports can be sent to the authors' email: diegesisandmimesis@proton.me.

Secret Questions

The following section contains secret questions. They work exactly like all the other hints in this booklet, only the question is hidden as well as the answers. We do this for things where the question itself would be spoiler-y.

Only develop the questions when directed to do so. You'll be directed here by other hints in this booklet, and occasionally when you seek out hints in-game. If you're told something like, "Develop secret question #1", find the specified question in this section, develop it, and then proceed as you would for any other hint in this booklet.

You can go ahead and develop secret question #1 right now if you want. It's a sample to illustrate how it works.
